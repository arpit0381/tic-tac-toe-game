# tic-tac-toe-game
A modern implementation of the classic Tic Tac Toe game built with HTML, CSS, and JavaScript 🎮

## Features
* Play as X or O against another player or solo
* Responsive design for a seamless gaming experience on desktop and mobile devices 📱
* Simple and intuitive gameplay

## Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** None (client-side only)
* **Dependencies:** None

## Installation
1. Clone the repository: `git clone https://github.com/your-username/tic-tac-toe-game.git`
2. Open the `index.html` file in a web browser

## Usage
* Open the `index.html` file in a web browser
* Click on a square to place your mark (X or O)
* Play against another player or solo and try to get three in a row to win 🏆
